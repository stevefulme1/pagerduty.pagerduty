# Code of Conduct

This project follows the [Ansible Community Code of Conduct](https://docs.ansible.com/ansible/devel/community/code_of_conduct.html).

## Summary

- Be respectful and considerate
- Be collaborative
- Be patient and generous
- Assume good intentions
- Try to be concise

## Reporting

If you believe someone is violating the code of conduct, please report it by
emailing the maintainers or opening a private issue.

For the full Ansible Community Code of Conduct, see:
https://docs.ansible.com/ansible/devel/community/code_of_conduct.html
